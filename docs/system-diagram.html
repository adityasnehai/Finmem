What FinMem Is vs Is Not

Is: A financial historian with episodic memory — retrieves and cites real historical market episodes structurally similar to current conditions.
Is: A research tool — surfaces base rates, historical context, regime transitions with explicit source attribution.
Is not: A trading signal or alpha generator. No price predictions, no buy/sell recommendations.
Is not: A general-purpose financial chatbot. Grounded exclusively in its episodic memory — refuses when uncertain.
Is not: A fine-tuned model. Knowledge lives in the memory store, not model weights — it is updatable, auditable, and citeable.
